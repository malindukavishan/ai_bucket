# AI BUCKET

AI BUCKET is a modern, curated directory of Artificial Intelligence tools. It helps users discover the best AI resources for Chat, Image generation, Writing, Coding, Video, Audio, and Productivity.

## Features

*   **Comprehensive Database**: Over 100+ AI tools categorized and tagged.
*   **Smart Filtering**: Filter tools by **Category**, **Platform** (Desktop/Mobile), and **Pricing** (Free, Freemium, Paid).
*   **Modern UI/UX**:
    *   Glassmorphism aesthetic.
    *   **Dark/Light Mode** support with persistent preference saving.
    *   **Neon Glow Animations** in Dark Mode.
    *   **Glitch Effect** text animations.
    *   **Dynamic Backgrounds**: Floating logos that drift across the screen.
*   **Interactive Cursor**: A glowing cursor follower in dark mode.
*   **Responsive Design**: Fully optimized for desktops, tablets, and mobile devices.

## Tech Stack

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom styling, CSS Variables for theming, Keyframe animations.
*   **JavaScript (Vanilla)**: Efficient DOM manipulation, filtering logic, and state management.
*   **Ionicons**: High-quality icons.

## How to Run

Since this project uses Vanilla web technologies, no build step is required.

1.  Clone or download the repository.
2.  Open `index.html` in your favorite web browser.

## Customization

*   **Add Tools**: Edit `js/main.js` and add new tool objects to the `aiTools` array.
*   **Change Theme Colors**: Modify the CSS variables (e.g., `--accent-primary`) in `css/style.css`.
*   **Adjust Animations**: Tweak keyframes in `css/animation.css` or `css/neon.css`.
